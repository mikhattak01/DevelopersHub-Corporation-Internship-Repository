# DevelopersHub-Corporation-Internship-Repository

## Overview
This project performs exploratory data analysis (EDA) and visualization on the classic Iris dataset. The analysis includes various plots to understand relationships between different features of Iris flowers and how they distinguish between three species: Iris-setosa, Iris-versicolor, and Iris-virginica.

## Libraries Used
The following Python libraries were imported and used in this analysis:

- **pandas** - For data manipulation and analysis
- **numpy** - For numerical operations
- **matplotlib** - For creating static visualizations
- **seaborn** - For statistical data visualization

The project includes the following visualizations:

2. **Box Plots**

## Key Findings Summary

Summary of the visualization is also provided in the jupyte notebook. It shows that the Iris dataset is well-structured with clear species separation, particularly when using petal features. The absence of outliers and clean distributions make it an excellent dataset for beginner data science projects and classification algorithm demonstrations.
